A full-stack web application for managing files on a CDN server.
The application allows users to upload, download, preview, and delete files.
Some features require authentication via a backend server connected to a database.

Built with:
  * Core language
    - JavaScript

  * Frontend 
    - HTML5 / CSS3
    - Bootstrap
    - Vue3
    - Vue Router
    - Axios

  * Backend 
    - Node.js 
    - Express
    - dotenv
    - pg (PostgreSQL client)
    - Cors

  * Authentication
    - JSON Web Token (JWT)
    - Password hashing
    - Protected API routes

Key Features:
  - Admin authentication
  - View list of photos
  - Upload photos one at a time or in butches
  - Delete photos
  - Download photos
  - Secure token-based communication with CDN API
